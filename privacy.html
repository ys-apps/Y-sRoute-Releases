プライバシーポリシー

公開主体: Y's

公開準備中

アカウントとクラウドデータのセルフサービス機能を実装・検証後に正式制定します。

1. 適用範囲

本ポリシーは、Y'sが提供するWindowsデスクトップアプリ「Y'sRoute」と、その公式案内ページにおける情報の取扱いに適用します。

2. 取り扱う情報

端末内に保存する情報

  • ゲーム、Todo、チャート、実行履歴、ゲームDB、チェックリスト、設定等の利用者入力データ
  • 利用者が選択した画像、アイコン、添付ファイルとバックアップ
  • 認証セッション（アクセストークン、更新トークン等）
  • ユーザーID、表示名、権限情報のプロフィールキャッシュ
  • 送信に失敗したXPイベントの再送用キュー

クラウドへ送信する情報

  • Supabase Authへ送信するメールアドレス、パスワードおよび認証に必要な情報。パスワードをY'sRoute独自のローカルDBへ保存しません。
  • ユーザーID、表示名、権限情報
  • XPイベントのアプリ識別子、操作分類、XP量、対象種別・対象ID、発生日時、重複防止ID
  • アカウント全体およびY'sRouteの進捗取得に必要なユーザー識別情報

公式案内ページ

GitHub Pagesへのアクセス時、GitHubがセキュリティ等の目的でIPアドレス等を処理する場合があります。Y'sRoute独自のアクセス解析、広告SDK、自動クラッシュレポートは初回リリースでは使用しません。

問い合わせフォーム

Googleフォームを通じて、対象アプリ、お問い合わせ種別、お問い合わせ内容、同意状況、および利用者が任意で入力した返信先メールアドレスを取得します。ファイル添付は受け付けません。

3. 利用目的

  • アプリ機能、ローカル保存、バックアップおよび復元の提供
  • 本人認証、ログイン状態の維持、アカウント管理
  • XP付与、重複送信防止、進捗表示
  • 問い合わせへの対応と、本人による開示・訂正・削除操作の提供
  • 不正利用防止、セキュリティ確保、障害調査

4. 保存先と第三者サービス

保存・送信先 内容 位置づけ
利用者のWindows端末 主要データ、画像、添付、設定、バックアップ、認証セッション、XPキュー 端末内保存
Supabase 認証、プロフィール参照、XPイベント、進捗 サービス提供のための委託先
GitHub 公式案内・法務文書の配信とアクセスログ 公開基盤
Google 問い合わせフォームへの入力内容と任意の返信先メールアドレス 問い合わせ受付のための委託先

法令に基づく場合を除き、利用者の情報を広告目的で販売しません。サービス提供に必要な委託先以外の第三者へ、本人の同意なく個人データを提供しません。

5. 保存期間

端末内データは利用者が削除するまで保存されます。クラウド情報はアカウント提供中または利用目的に必要な期間保存し、削除依頼の完了後も、法令上の保存義務、セキュリティログ、サービス事業者のバックアップローテーションに必要な期間残る場合があります。

6. 安全管理

認証情報へのアクセス制限、通信の暗号化、利用者単位のアクセス制御、必要最小限の情報送信、秘密情報を配布物へ含めない運用等の措置を講じます。

7. 利用者の請求と削除

開示、訂正、利用停止、削除、アカウント削除はデータの開示・削除手順をご確認ください。ローカルデータとクラウドデータは保存先が異なるため、それぞれの削除が必要です。

8. 未成年者

未成年者は、必要に応じて保護者等の法定代理人の同意を得て本サービスを利用してください。

9. 改定

取扱内容を変更する場合は本ページを更新し、重要な変更は公式案内またはアプリ内で告知します。

10. 問い合わせ先

アカウント・クラウドデータの削除、開示、訂正、利用停止の請求は、この一般問い合わせフォームでは受け付けていません。専用手順はデータの開示・削除手順をご確認ください。
